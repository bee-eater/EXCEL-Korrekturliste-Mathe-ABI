# Changelog

Alle wichtigen Änderungen in diesem Projekt werden in diesem Changelog dokumentiert.

## [2.3.0] - Not released yet

### 2.3.0 - Geplante Features

- Möglichkeit ein Update direkt aus EXCEL heraus zu starten. [#4]
- **Weitere Verbesserungsvorschläge sind jederzeit willkommen!**

## [2.2.0] - 2026-05-05

### 2.2.0 - Neue Features

- Möglichkeit für Zweit- und Drittkorrektor hinzugefügt [#1]
- Optionen für leichteres Eintragen hinzugefügt

### 2.2.0 - Verwendung

- ZK/DK kann in der Config angelegt werden.
- Zeilen für ZK/DK können jederzeit hinzugefügt und gelöscht werden. Beim Löschen gehen logischerweise eingetragene Punkte des ZK/DK verloren!
- Es gibt vier Buttons über welche die Ansicht für den jeweils Korrigierenden umgeschaltet werden kann. Bspw. werden für den "EK" == Erstkorrektor nur die eigenen Zeilen eingeblendet, so, als gäbe es die anderen gar nicht! ZK/DK sehen grau die Einträge des EK. "All" blendet alles ein.
- Zusätzlich kann über ZK / DK importieren, eine Datei mit !identischer Konfiguration! importiert werden. Dieser Import passiert immer in der Hauptliste! Wenn der ZK/DK die Liste vom EK übernehmen will, dann muss er dessen Liste öffnen und die eigene Liste dort importieren. Dann kann er mit dieser Version weiterarbeiten und die alte Liste verwerfen!

## [2.1.0] - 2026-03-30

### 2.1.0 - Neue Features

- Wahlaufgaben können jetzt angelegt werden

### 2.1.0 - Verwendung

- Wie gewohnt Schüler und Aufgaben anlegen
- Wahlaufgaben werden nicht beim Teilbereich, sondern in einem extra Bereich hinterlegt
- Tabellen erstellen
- Es gibt einen neuen Tab "ConfigW", hier werden die Aufgaben, welche der Schüler ausgewählt hat, mit "x" selektiert
- Button "Wahlaufgaben aktualisieren" auf der Config-Seite drücken, um das Wahlaufgaben-Tab zu aktualisieren
